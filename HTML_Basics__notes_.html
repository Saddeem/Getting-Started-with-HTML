Main Concepts: HTML Basics

1. Introduction
HTML Stands for Hyper Text Markup Language. It consists of text content, markup, references to other documents (like images or videos) and/or references to other pages.

2. HTML Tags
HTML tags are a way of changing how the texts looks on a webpage. It consists of an opening tag, followed by the content, which is what the tag is working on, then a closing tag.

They are many different kinds of tags and each serve a different function. For example, to make a sentence bold, the < b > tag can be used. Another example is the italics tag, in which < em > is used instead of < b > making the content here italicized instead. It’s important to note that if we don’t close the tag, all content after the opening tag will be bold or italicized, and that’s the case with most tags. 3. Inline Vs Block Elements
Inline tags are tags that can be placed within text without creating a line break. (like < b >, < em > and even < img >) Block tags, on the other hand, contain the text in an invisible box, hence the name block. The < p > tag, which stands for paragraph, wraps the text within a box that separates it from any other text surrounding it. The < div > tag works in a similar manner as well.
